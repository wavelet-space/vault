Qt¶

  • Kompilace knihovny Qt5 získané z GitHub repozitáře.

    • git clone –recurse-submodules

    • git git submodule init/deinit

    • perl init-repository –module-subset=qtbase,qtshadertools,qtdeclarative

Toto mlže tvat hodně dlouho, protože se stahuje několig GB dat. Pokud chceme jen podmnožinu můžeme omezit stahované repozitáře

Licence Qt:

  • https://qr.ae/pypdnh

Zásadní zdroj je tento https://wiki.qt.io/Building_Qt_6_from_Git

Qt můžeme stáhnout přímo z GitHub repozitáře, který se jmenuje qt5 avšak obsahuje mnoho větví např. i 6.5 atd. Takže pokud chceme verzi 6 musíme se přepnout do požadované verze.
